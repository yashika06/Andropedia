Option Menu

The options menu appears when the user presses the menu button on their Android device. The options menu is typically used for providing additional info about an app, as well as linking to a help and settings sections. To implement an options menu for an Activity in an Android app, a few fairly straightforward steps are required.

  • Creating a menu XML file Create a menu xml file by right clicking on menu folder inside res folder. 
                        <menu xmlns:android="http://schemas.android.com/apk/res/android">

                    </menu>
    Between these opening and closing tags, we will place XML markup for each item in our options menu.
  • Add items to our menu Add an item for each menu option using the following syntax: 
                        <item android:id="@+id/about"
                          android:icon="@drawable/about"
                          android:title="About" />
                    <item android:id="@+id/help"
                          android:icon="@drawable/help"
                          android:title="Help" />
  • Inflating menu resource To instruct Android to use our options menu, open the Activity class we want it to appear with. Add the following method to our Java code 
                        @override
                    public boolean onCreateOptionsMenu(Menu menu) {
                        MenuInflater inflater = getMenuInflater();
                        inflater.inflate(R.menu.my_options_menu, menu);
                        return true;
                    }
  • Respond to menu item selection Before our code can respond appropriately to user interaction with the menu, we need to work out which item was selected. Add a switch statement to our method using the following sample syntax: 
                        public boolean onOptionsItemSelected(MenuItem item) {
                        switch (item.getItemId()) {
                            case R.id.about:
                            startActivity(new Intent(this, About.class));
                            return true;
                            case R.id.help:
                            startActivity(new Intent(this, Help.class));
                            return true;
                            default:
                            return super.onOptionsItemSelected(item);
                        }
                    }
