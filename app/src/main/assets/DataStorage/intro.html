Introduction

Data storage is used to store application data in databases, files, or preferences, in internal or removable storage. Android provides several options to save persistent application data. The option choosen depends on the specific need, such as whether the data should be private or accessible to other applications.

The data storage options are as follows:

  • Shared Preferences Store private primitive data in key-value pairs.
  • Internal Storage Store private data on the device memory.
  • External Storage Store public data on the shared external storage.
  • SQLite Databases Store structured data in a private database.
