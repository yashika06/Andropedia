Creating a fragment using XML

Various steps require to create a fragment using XML are:

  • Create a layout file that contain the appearance of fragment 
                        <?xml version="1.0" encoding="utf-8"?>
                    <FrameLayout xmlns:android="http://schemas.android.com/apk/res/android"
                        xmlns:tools="http://schemas.android.com/tools"
                        android:layout_width="match_parent"
                        android:layout_height="match_parent"
                        tools:context=".MyFragment"
                        android:id="@+id/fragment">

                       <TextView android:id="@+id/text"
                          android:layout_width="wrap_content"
                          android:layout_height="wrap_content"
                          android:text="This is a TextView" />
                    </FrameLayout>
  • Create a class that extends Fragment 
                        public class MyFragment extends Fragment{
                        public View onCreateView(LayoutInflater inflater, ViewGroup container, Bundle savedInstanceState){
                            return inflater.inflate(R.layout.fragment_my, container, false);
                        }
                    }
  • Attach the fragment inside Activity through XML 
                        <fragment
                         android:id="@+id/fragment_container"
                         android:layout_width="match_parent"
                         android:layout_height="wrap_content"
                         android:name=".MyFragment"/>
