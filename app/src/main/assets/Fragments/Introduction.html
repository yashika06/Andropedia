Introduction

A Fragment represents a behavior or a portion of user interface in an Activity. You can combine multiple fragments in a single activity and reuse a fragment in multiple activities. Interaction with fragments is done through FragmentManager. Fragments represents a particular operation or interface that is running within a larger Activity.

Though Fragment defines its own lifecycle, that lifecycle is dependent on its activity: if the activity is stopped, no fragments inside of it can be started; when the activity is destroyed, all fragments will be destroyed.
