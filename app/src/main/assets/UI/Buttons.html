Buttons

Android Button represents a push-button. The android.widget.Button is subclass of TextView class and CompoundButton is the subclass of Button class. A button consists of text or an icon (or both text and an icon) that communicates what action occurs when the user touches it.

  • A button with text only, using Button class. 
                       <Button android:id="@+id/button"
                      android:layout_width="wrap_content"
                      android:layout_height="wrap_content"
                      android:text="@string/button_text" />
  • A button with icon only, using ImageButton class. 
                       <ImageButton android:id="@+id/image_button"
                      android:layout_width="wrap_content"
                      android:layout_height="wrap_content"
                      android:src="@drawable/button_icon" />
  • A button with text and icon, using Button class with the android:drawableLeft attribute. 
                       <Button android:id="@+id/button"
                      android:layout_width="wrap_content"
                      android:layout_height="wrap_content"
                      android:text="@string/button_text"
                    android:drawableLeft="@drawable/button_icon" />

Buttons click events

Click event occurs when a user click on a particular button. On button click some action should be performed which is defind in click event. There are two ways of

  • Using onClick attribute
    Here's the layout with a button using android:onClick 
                       <Button android:id="@+id/button"
                      android:layout_width="wrap_content"
                      android:layout_height="wrap_content"
                      android:text="@string/button_text"
                      android:drawableLeft="@drawable/button_icon"
                      android:onClick="buttonClicked"/>
    Within the Activity that hosts this layout, following method handles the click event: 
                       /** Called when the user touches the button */
                    public void buttonClicked(View view) {
                        // Do something in response to button click
                    }
  • Using onClickListener
    To declare the event handler programmatically, create an View.OnClickListener object and assign it to the button by calling setOnClickListener. 
                       Button button = (Button) findViewById(R.id.button_send);
                    button.setOnClickListener(new View.OnClickListener() {
                        public void onClick(View v) {
                            // Do something in response to button click
                        }
                    });
