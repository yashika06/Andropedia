Toast

A toast is a view containing a quick little message for the user. The toast class helps you create and show those. When the view is shown to the user, appears as a floating view over the application.

The following code is used to display a toast:

                        Toast.makeText(getApplicationContext(),"Hello Programmers",Toast.LENGTH_SHORT).show();

Here, getApplicationContext() method returns the instance of Context.

We can also set margins to the toast.

                        Toast toast=Toast.makeText(MainActivity.this,"Hello Programmers",Toast.LENGTH_SHORT);
                        toast.setMargin(20,20);
                        toast.show();
