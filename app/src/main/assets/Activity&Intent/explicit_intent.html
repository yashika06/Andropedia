Explicit Intent

An explicit intent is one that we use to launch a specific app component, such as a particular activity or service in our app. Explicit Intents specify a component (via setComponent(ComponentName) or setClass(Context, Class)), which provides the exact class to be run. We can also pass the information from one activity to another using explicit intent.

                        Intent i = new Intent(getApplicationContext(), NextActivity.class);
                        i.putExtra("EXTRA_STRING", "Hello next Activity");
                        startActivity(i);

In the above code snippet we are starting NextActivity and at the same time sending data using putExtra() method. To retrieve data in NextActivity following code is used: 

                        Intent intent = getIntent();
                        String str = intent.getStringExtra("EXTRA_STRING");
